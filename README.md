# Semantic Search Engine (Embeddings + Vector Similarity)

This project demonstrates how semantic search works using embeddings and cosine similarity.

---

## 🚀 Features

- Convert text into embeddings using Gemini API  
- Store vectors in memory  
- Perform semantic search using cosine similarity  
---

## 🧠 Concepts Covered

- Embeddings (text → vectors)  
- Vector similarity (cosine similarity)  
- High-dimensional vector space  
- Semantic search
